# Code Reader - 项目文档

## 项目概述

Code Reader 是一个本地代码阅读与分析工具。它提供直观的文件浏览、智能代码高亮、全文检索和 AI 辅助代码理解功能，帮助开发者快速理解和分析项目代码。

---

## 核心功能

### 1. 本地项目目录解析与文件树展示

**功能描述**：
- 自动扫描并展示本地项目的完整目录结构
- 支持文件夹展开/折叠导航
- 智能过滤非代码文件（图片、二进制文件等）
- 实时监听文件变化，自动更新文件树

**技术实现**：
- 前端：React + Vite 的 `import.meta.glob` 动态导入
- 后端：Node.js + Express 文件系统 API
- 支持热更新（HMR）

**文件类型支持**：
- 源代码：`.ts`, `.tsx`, `.js`, `.jsx`, `.cpp`, `.c`, `.h`, `.java`, `.py`, `.go`, `.rs` 等
- 样式文件：`.css`, `.scss`, `.less`
- 配置文件：`.json`, `.yaml`, `.yml`, `.toml`, `.env`
- 文档：`.md`, `.txt`

---

### 2. 多语言代码语法高亮

**功能描述**：
- 自动识别文件类型并应用对应的语法高亮
- 支持 100+ 种编程语言
- 提供多种主题（VS Code Dark、GitHub Light 等）
- 支持代码折叠、行号显示、括号匹配

**技术实现**：
- 使用 `Prism.js` 或 `highlight.js` 进行语法高亮
- 集成 Monaco Editor 或 CodeMirror 作为代码编辑器
- 按需加载语言包，优化性能

**支持的语言**：
- JavaScript/TypeScript
- Python
- Java
- C/C++
- Go
- Rust
- HTML/CSS
- JSON/YAML
- 等 100+ 种语言

---

### 3. 关键词/函数全文检索

**功能描述**：
- 在整个项目中搜索关键词、函数名、变量名
- 支持正则表达式搜索
- 搜索结果高亮显示
- 点击结果直接跳转到对应代码位置
- 支持过滤文件类型

**技术实现**：
- **前端方案**（小型项目）：
  - 使用 `Fuse.js` 或 `flexsearch` 进行客户端搜索
  - 适合文件数量 < 1000 的项目
  
- **后端方案**（大型项目，推荐）：
  - 使用 Elasticsearch 或 Meilisearch
  - 建立代码索引（函数名、变量名、注释等）
  - 支持模糊匹配、代码片段定位

**搜索功能**：
- 精确匹配
- 模糊搜索
- 正则表达式
- 按文件类型过滤
- 搜索结果分组展示

---

### 4. AI 辅助解释代码、生成注释

**功能描述**：
- 选中代码块，一键获取 AI 解释
- 自动生成函数注释和文档
- 代码优化建议
- 代码重构建议
- 支持多轮对话，深入理解代码逻辑

**技术实现**：
- **AI 模型选择**：
  - OpenAI GPT-4 / Claude（效果好，付费）
  - 通义千问 / 文心一言（国内 API，成本较低）
  - CodeLlama / StarCoder（本地部署，免费）
  
- **交互方式**：
  - 右键菜单："解释这段代码"、"生成注释"
  - 侧边栏 AI 助手面板
  - 快捷键触发
  
- **Prompt 模板**：
  - 代码解释模板
  - 注释生成模板
  - 代码审查模板

**AI 功能列表**：
1. **代码解释**：用自然语言解释代码逻辑
2. **注释生成**：为函数/类生成标准注释
3. **代码优化**：提供性能优化建议
4. **Bug 检测**：识别潜在问题
5. **代码重构**：建议更好的代码结构
6. **文档生成**：生成 API 文档

## 技术架构

### 前端技术栈

```
React 19
├── Vite（构建工具）
├── TypeScript（类型安全）
├── Monaco Editor / CodeMirror（代码编辑器）
── Prism.js / highlight.js（语法高亮）
├── Fuse.js / flexsearch（前端搜索）
└── React Router（路由管理）
```

### 后端技术栈

```
Node.js + Express
├── 文件系统 API（读取本地文件）
├── Elasticsearch / Meilisearch（全文检索）
├── WebSocket（实时文件监听）
└── AI API 代理（OpenAI / 通义千问）
```

### 项目结构

```
code-reader/
├── src/
│   ├── components/
│   │   ├── FileTree/          # 文件树组件
│   │   ├── CodeEditor/        # 代码编辑器
│   │   ├── SearchPanel/       # 搜索面板
│   │   ├── AIAssistant/       # AI 助手面板
│   │   └── Toolbar/           # 工具栏
│   ├── hooks/                 # 自定义 Hooks
│   ├── utils/                 # 工具函数
│   ├── styles/                # 样式文件
│   ├── App.tsx                # 主应用组件
│   └── main.tsx               # 入口文件
├── server/                    # 后端服务（可选）
│   ├── api/                   # API 路由
│   ├── search/                # 搜索索引
│   └── ai/                    # AI 服务
├── public/                    # 静态资源
├── package.json
└── vite.config.ts
```
